# ABOUT XYLFTP

* Description: Xiyou Linux FTP Project
  * XYLFTP is an FTP suit that contains both an FTP client and an FTP server.
  * Reference: https://datatracker.ietf.org/doc/html/rfc959
  * The project was hosted at sourceforge.net(CVS) and switched to repo.or.cz in June 2007.
* Original Homepage URL: https://sourceforge.net/projects/xylftp
* Original Git Repo
  * git://repo.or.cz/xylftp.git
  * https://repo.or.cz/xylftp.git
